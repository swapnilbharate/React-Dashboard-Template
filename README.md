A modern and responsive dashboard built using React.js, Redux Toolkit, React Router, Recharts, and Vite.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/swapnilbharate/React-Dashboard-Template.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
