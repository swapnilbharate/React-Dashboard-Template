# 🚀 React Dashboard Template

A modern and responsive dashboard built using React.js, Redux Toolkit, React Router, and Vite.

## ✨ Features

- Dashboard Analytics
- User Management
- Reports Section
- Notifications
- Profile Page
- Responsive Design
- Sidebar Navigation
- Redux Toolkit State Management

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- React Router DOM
- Recharts
- Vite
- CSS3
